# INSTALAR Y EJECUTAR EL PROYECTO

Desde la terminal, dirigirse a la carpeta donde quieras instalar el proyecto y ejecuta...

```bash
  # clonar carpeta
  git clone https://github.com/Yiosko/HojasDeVidaPHP.git

```

Ingresar a la carpeta del proyecto

```bash
  # carpeta proyecto-hoja-vida
  cd .\proyecto-hoja-vida\
```


```bash
  # ejecutar puerto 8000 localhost
  php artisan serve
```


Y listo en el navegador, ingresar al host

```
  http://127.0.0.1:8000/
```
